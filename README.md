# Student Result Management System

This is a simple Python project to calculate student marks, average and grade.

## Features
- Takes student input
- Calculates total marks
- Calculates average
- Assigns grade

## Language
Python

## Sample Output
Enter student name: Hani
Enter marks in Subject 1: 85
Enter marks in Subject 2: 90
Enter marks in Subject 3: 88

Name: Hani
Total: 263
Average: 87.6
Grade: B
